Formatação de HD

Realizar a formatação de uma unidade (HD, SSD, unidade flash etc.) significa preparar a partição escolhida na unidade a ser usada por um sistema operacional, excluindo todos os dados e configurando um sistema de arquivos . O sistema de arquivos mais popular para oferecer suporte ao Windows é o NTFS, mas o FAT32 também é usado algumas vezes. No Windows, a formatação de uma partição geralmente é feita na ferramenta Gerenciamento de Disco. Você também pode formatar uma unidade usando o comando format em uma interface de linha de comando como o Prompt de Comando ou com uma ferramenta de software de partição de disco livre.

Instalação de S.O

Em computação, uma instalação (do inglês installation ou setup) é um processo (ou seu resultado) quando todos os arquivos necessários são colocados num computador para que o programa (por exemplo, sistema operacional, driver de dispositivo, software aplicativo, módulo de extensão, patch, etc.) possa ser executado. Tais computadores podem ser smartphones, computadores pessoais, servidores, etc. No entanto, o termo "instalação" também usado no caso quando os arquivos colocados não são necessários por qualquer programa particular e a instalação é apenas uma das maneiras para armazenar e gerenciar os arquivos pelo usuário. Alguns exemplos: bibliotecas de exemplos (projetos) tais como imagens vetoriais, código fonte, vários modelos (por exemplo, CAD, CASE, processos, etc.), bibliotecas de multimídia tais como livros digitais, apresentações, cursos, etc.

Crimpagem de cabo de rede

Crimpagem é, basicamente, a montagem de um cabo de rede. Por inúmeras razões, nem sempre é possível comprar um cabo pronto, sendo necessário montar um conforme as suas necessidades, como conectar aparelhos distantes do roteador.

Hoje em dia, é bem comum que existam redes de computadores — e não somente em grandes organizações, onde a rede pode ser composta por uma grande quantidade de computadores, mas também em residências, em que pode existir uma rede de apenas duas máquinas, por exemplo.

Conserto de Controle de PS4

Controle de Playstation 4, conforme o uso, vai se deteriorando ao longo do tempo. Há necessidade de ser realizada uma manutenção preventiva ou corretiva desse controle.

Geralmente um controle original dura mais tempo do que um paralelo, devido a qualidade de fabricação do original ser bem melhor.

Limpeza interna de PS4

A limpeza interna e a troca da pasta térmica do Playstation 4 é extremamente necessária.Se o uso for constante o video-game fica com mais sujeira ao longo do tempo.

Recomendo essa manutenção preventiva a cada 18 meses.Se o PS4 ficar alguns dias sem uso, indico colocar uma capa protetora para protegê-lo de eventuais sujeiras.

Configuração de roteador

Os roteadores orientam e direcionam os dados da rede, usando pacotes que contêm vários tipos de dados, como arquivos e comunicações e transmissões simples, como interações na Web.

Os pacotes de dados têm várias camadas ou seções, uma das quais contém informações de identificação, como remetente, tipo de dados, tamanho e, o mais importante, o endereço IP de destino (protocolo de Internet). O roteador lê essa camada, prioriza os dados e escolhe a melhor rota a ser usada para cada transmissão.

Instalação de Antena UHF

De acordo com a Portaria N°378/2016 do Ministério das comunicações, até o ano de 2023, o desligamento do sinal de TV analógico deve acontecer em todas as cidades do país. Por isso, desde então, algumas regiões já começaram a utilizar a antena para TV digital. O novo sistema não altera a transmissão em alta definição e a recepção portátil já é usada pelo antigo sinal. Entretanto, ele proporciona um grande avanço de imagem e som para os telespectadores.
